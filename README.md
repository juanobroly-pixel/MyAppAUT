# 🚀 Universal Android Recovery & Flash Tool

![Build Status](https://github.com/juanobroly-pixel/My-App-AUT/actions/workflows/distribucion.yml/badge.svg)

## 📱 Visión General
Una herramienta de código abierto diseñada para simplificar el rescate, actualización y mantenimiento de dispositivos Android (como Samsung y Motorola) con un solo toque.

## ✨ Características Universales
* **Agnóstico de Marca**: Soporte para protocolos universales ADB y Fastboot.
* **Smart Flash**: Detección automática de hardware y niveles de binario (como el Binario 6).
* **Entrega Continua**: Automatización total vía GitHub Actions y Firebase App Distribution.
